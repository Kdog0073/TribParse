TRIB_PARSER README

REQUIREMENTS:
Python 2.7
Mozilla Firefox
Selenium (command prompt: pip install selenium)

PYTHON:
trib_parser: original coding
trib_parser_full: parser for all cases
trib_parser_rand_tenth: parser for ~1/10 cases at random
trib_parser_rand_hundreth: parser for ~1/100 cases at random
trib_parser_rand_thousandth: parser for ~1/1000 cases at random
trib_parser_test: parser for testing functionality
trib_parserXX: parsers for 1/25th of the data. You can run multiple of these in parallel. (These all refer to trib_parser in the code).

DOCUMENTS
/Resources/ErrorURL: a document that documents all urls that gave errors
/Results/Defeat_Chat/*: documents containing defeated team chat
/Results/Victory_Chat/*: documents containing victorious team chat
/Sample/tribunal html sample.pdf: a commented pdf showing a sample tribunal case
/Sample/Results/*: Sample results
